# AGENTS.md

## Project Context

This folder contains the frontend prototype for StudyAgent, a college learning-management tool-calling AI agent.

The frontend is currently a static demo. It should not call a real backend yet.

## Tech Stack

Use only:

* HTML
* CSS
* Vanilla JavaScript

Do not use:

* Vue
* React
* Vite
* npm
* TypeScript
* External CDN resources

## Files

Expected files:

* index.html
* style.css
* app.js

## UI Requirements

The page should show:

* Project header
* Sidebar menu
* Chat interface
* Tool call process panel
* Learning statistics dashboard
* Mock responses for demo questions

## Important Constraints

* Do not modify files outside this frontend folder.
* Do not create backend files.
* Do not add package managers.
* The page must run by directly opening index.html in a browser.
* Keep the code simple enough for first-year students to explain.

## Definition of Done

Before finishing, verify:

* index.html loads style.css and app.js correctly.
* Demo question buttons work.
* Chat messages appear.
* Tool call cards update.
* The page works without any server.
